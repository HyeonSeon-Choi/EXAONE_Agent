### 버전 1: 간단한 요청

```
안녕하세요 [디자이너 이름]님!

EXAONE AI Agent 앱의 디자인 작업을 부탁드리고 싶습니다.

현재 기본 레이아웃은 완성되어 있지만, 다음 항목들에 대한 
디자인이 필요합니다:

✅ 커스텀 아이콘 디자인 (3개)
✅ EXAONE 로고 디자인  
✅ UI 컴포넌트 상태별 디자인
✅ 전체적인 UI/UX 개선

Figma로 작업해주시면 되고, 자세한 가이드는 
DESIGN_BRIEF.md 파일에 정리해두었습니다.

작업 기간은 [기간] 정도로 생각하고 있는데, 
일정 조율 가능하신가요?

감사합니다!
```

### 버전 2: 상세한 요청

```
안녕하세요 [디자이너 이름]님!

EXAONE AI Agent 프로젝트의 디자인 작업을 요청드립니다.

📱 프로젝트 개요
- 플랫폼: Android 앱
- 목적: 엔터프라이즈용 AI 에이전트 애플리케이션
- 디자인 도구: Figma

🎨 현재 상태
- 기본 레이아웃 및 색상 시스템은 구축 완료
- Material Design 3 기반으로 작업 중
- 로그인 화면, 메인 화면 기본 구조 완성

📋 요청 작업 내용

1. 아이콘 디자인 (우선순위: 높음)
   - NL-to-SQL 조회 아이콘
   - 자동 리포팅 아이콘
   - Voice Interface 아이콘
   - SVG 또는 벡터 형식 권장

2. 로고 디자인 (우선순위: 높음)
   - EXAONE 브랜드 아이덴티티 반영
   - 다양한 해상도 버전 필요

3. 컴포넌트 상태 디자인 (우선순위: 높음)
   - Normal, Focused, Disabled, Error 상태
   - 터치 피드백 효과

4. UI/UX 개선 (우선순위: 중간)
   - 전체적인 시각적 완성도 향상
   - 사용자 경험 개선

📐 작업 가이드
- DESIGN_BRIEF.md 파일에 상세 가이드 정리
- Android 개발을 위해 dp 단위 사용 필수
- 8dp 그리드 시스템 준수
- Material Design 3 가이드라인 준수

📅 일정
- 작업 기간: [기간]
- 중간 리뷰: [날짜]
- 최종 완료: [날짜]

💬 협업 방식
- Figma 파일 공유
- 중간 피드백 및 리뷰
- 개발자와 디자인 핸드오프

자세한 내용은 DESIGN_BRIEF.md 파일을 참고해주세요.
질문이나 추가 논의가 필요하시면 언제든 연락주세요!

감사합니다.
```
