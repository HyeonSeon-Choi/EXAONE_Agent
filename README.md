# EXAONE AI Agent (Android)

**EXAONE AI Agent**는 기업 환경에서 AI를 활용해 업무 효율을 극대화하는 안드로이드 애플리케이션입니다.

## 🚀 주요 기능 (Core Features)
* **NL-to-SQL 조회**: 자연어로 데이터베이스를 조회하고 분석 결과를 확인합니다.
* **자동 리포팅 & 요약**: 복잡한 데이터를 시각화하고 요약 리포트를 생성합니다.
* **Voice Interface**: 음성 인식 기능을 통해 손을 쓰지 않고 업무를 처리합니다.
* **사내 계정 시스템**: 사번 기반의 회원가입 및 부서/직급별 프로필 관리를 지원합니다.

## 🎨 UI/UX 디자인 가이드
* **디자인 시스템**: Google Material Design 3 (M3) 기반.
* **브랜드 컬러**: EXAONE Red (#E31E2C)를 메인 테마로 적용.
* **반응형 레이아웃**: `ConstraintLayout`과 `MaterialCardView`를 활용한 세련된 카드형 UI.

## 🛠 기술 스택 (Tech Stack)
* **Language**: Kotlin
* **SDK**: Compile SDK 36 / Min SDK 26
* **Features**: ViewBinding, Coroutines, Retrofit (준비 중)
* **Resources**: 부서별(IT개발, 데이터분석 등) 특화된 리소스 정의
